# フリーランス税金シミュレータ

フリーランス・個人事業主が年間売上と経費を入力するだけで、複数税目の内訳と手取り額を一括で試算できる静的SPAです。

## 対応税目（令和8年分）

- 所得税＋復興特別所得税
- 住民税（所得割＋均等割＋森林環境税）
- 個人事業税
- 国民年金・国民健康保険

## 技術スタック

- Vite + React + TypeScript
- テスト: Vitest
- バックエンドなし・外部通信なし（完全クライアント完結）

## セットアップ

```bash
npm install
npm run dev      # 開発サーバー起動
npm run test     # テスト実行
npm run build    # 本番ビルド
```

## ディレクトリ構成

```
src/
  data/      税制パラメータJSON・型定義
  logic/     計算エンジン（純関数、React非依存）
  ui/        画面描画（Reactコンポーネント）
```

## 免責事項

本シミュレーションは概算です。実際の税額は申告内容・自治体により異なります。
