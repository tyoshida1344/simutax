## 2. 国民健康保険（全国標準・賦課限度額）

| 項目 | 内容 |
|------|------|
| 管轄 | 厚生労働省 |
| URL | https://www.mhlw.go.jp/stf/newpage_21081.html |
| 検索キーワード | `国民健康保険 賦課限度額 令和○年度 厚生労働省` |
| 改定時期 | 毎年4月（前年12月の社会保障審議会で決定） |
| 取得データ | 医療分・後期高齢者支援金分・介護分の各賦課限度額（万円） |
| JSON箇所 | `socialInsurance.nationalHealthInsurance.models.standard.*.cap` |
| ページの見方 | 「国民健康保険の保険料（税）の賦課限度額」の表から3区分の上限額を読み取る。額は万円単位で記載されることが多いので、円に換算して入力（例: 67万円 → 670000） |

全国標準モデルの料率（`incomeRate`, `perCapita`, `perHousehold`）は厚生労働省の「国民健康保険事業年報」から取得できるが、実用上は都道府県庁所在地の料率を使うため、標準モデルの料率更新は優先度低。

## 3. 国民健康保険（47都道府県庁所在地）

| 項目 | 内容 |
|------|------|
| 管轄 | 各市区町村 |
| 改定時期 | 毎年4〜6月（市議会の議決後に公表） |
| 取得データ | 各市の所得割率・均等割額・平等割額・賦課限度額（医療分/後期高齢者支援金分/介護分） |
| JSON箇所 | `prefectures.*.nhi.{medical,elderlySupport,nursingCare}.{incomeRate,perCapita,perHousehold,cap}` |

### 検索方法

各市の公式サイトで以下を検索する:
```
{市名} 国民健康保険料 令和○年度
```

### 読み取る表の項目と対応

| 自治体の表記 | JSONフィールド | 注意 |
|-------------|---------------|------|
| 所得割率 | `incomeRate` | 小数に変換（例: 7.51% → 0.0751） |
| 均等割額（被保険者均等割） | `perCapita` | 円単位（1人あたり年額） |
| 平等割額（世帯別平等割） | `perHousehold` | 円単位。市によっては0（東京23区、さいたま市等） |
| 賦課限度額（課税限度額） | `cap` | 円単位。国の上限以下の場合がある |

### 注意事項
- 「医療分」「基礎分」「一般分」は同じもの（= `medical`）
- 「後期高齢者支援金分」「支援分」は同じもの（= `elderlySupport`）
- 「介護納付金分」「介護分」は同じもの（= `nursingCare`）
- 介護分の `minAge: 40`, `maxAge: 64` は法定なので変更不要
- 一部の市は「保険税」方式だが、構造は同じ（率・額の名前が異なるだけ）

| JSON箇所 | `residentTax` |

### 都道府県別の超過課税

一部の自治体は標準税率に上乗せしている:
- 名古屋市: 市民税が5.3%に軽減（県民税2%と合わせて所得割9.7%を想定していたが、令和8年時点で軽減廃止の方向もあるので要確認）
- 横浜市: みなと横浜ボートレース振興会分の水源環境保全税等で所得割10.025%
- その他: 森林環境税・水源環境税等で均等割が異なる都道府県がある

| JSON箇所 | 説明 |
|----------|------|
| `prefectures.*.residentTaxIncomeRate` | 都道府県＋市区町村の合計所得割率 |
| `prefectures.*.residentTaxPerCapitaLevy` | 均等割（都道府県＋市区町村＋森林環境税） |
